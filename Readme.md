## Configuration

<!--modlib:conf:2-->
### `liquid_dir`

Show directions of all flowingliquid nodes

* Type: boolean
* Default: `false`

### `liquid_raycast`

Show results of modlib's accurate liquid raycast

* Type: boolean
* Default: `false`
